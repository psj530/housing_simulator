# 둘이서 한 집 — 예비신혼부부 시뮬레이션 허브

예비신혼부부(홍지훈 · 박수정)를 위한 **단일 HTML 웹앱**입니다.
결혼 준비부터 내 집 마련(대출)까지 한 화면에서 슬라이더·직접입력으로 계산하는 **참고용 도구**예요.
백엔드 없이 브라우저에서만 계산하며, 개인정보는 서버로 전송되지 않아요. (모든 금액 단위 = 천원)

## 실행 방법
`index.html`을 브라우저로 열면 끝이에요. 별도 빌드/설치 불필요.

### GitHub Pages 배포
1. 이 저장소를 GitHub에 push
2. **Settings → Pages → Branch: main / (root)** 선택 후 저장
3. 잠시 뒤 `https://<사용자명>.github.io/<저장소명>/` 에서 접속

## 구성
- `index.html` — 웹앱 본체 (HTML + CSS + JS 단일 파일)
- `이관요약_예비신혼부부_시뮬레이션_허브.md` — 프로젝트 이관/개발 요약 문서

## 탭 구조 (6개)
내 집 마련(기본) → 결혼 준비 → 하객 → 신혼여행 → 상견례 → 종합

## 주요 계산 로직
- 대출: 취득세 · LTV · 6억 cap · DSR · 스트레스 금리(2026) · 생애최초(70/40)
- 상환 3방식: 원리금균등 · 원금균등 · 체증식
- 통근 거리(하버사인) · 관심매물 22곳 · 지도/실거래 링크

## 참고
계산 결과는 참고용이며, 실제 대출 한도·세금은 금융기관·관할 기준에 따라 달라질 수 있어요.
